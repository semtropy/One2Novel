# One2Novel 验收清单

每次改动后，打开 **http://localhost:7457** 逐项走一遍。

---

## 环境

```bash
pnpm dev             # server:7456 + client:7457
pnpm typecheck       # 零错误
```

---

## 首页

- [ ] 输入灵感 → "开始创作" → AI 生成故事种子 → "进入创作工作台"
- [ ] 已有灵感时 Enter 快捷键直达生成
- [ ] "已有小说？进入我的小说" 链接可用

## 顶栏导航

- [ ] Logo 点击回首页
- [ ] 我的小说 / 写法引擎 / 创作助手 三个入口
- [ ] 右上角设置齿轮

## 规划区

### 故事种子
- [ ] AI 生成填充字段
- [ ] 前提/主线/悬念/结局 失焦自动保存
- [ ] 题材/视角/节奏/情感 下拉可改

### 角色阵容
- [ ] AI 生成角色 + 手动添加角色
- [ ] 角色卡片三栏排列（标签 + 名字 + 背景摘要）
- [ ] 点击卡片打开弹窗，所有字段失焦自动保存
- [ ] 外貌档案在弹窗内展开，AI 生成可用
- [ ] 关系图：左 SVG + 右关系列表，彩色连线，增删关系

### 章节蓝图
- [ ] AI 生成蓝图（写 DraftPlan，不影响写作区）
- [ ] 手动 + 卷 + 章，标题/摘要失焦自动保存
- [ ] 删卷/删章（不影响写作区）
- [ ] 撤销按钮：将蓝图回退到写作区当前结构
- [ ] 确认按钮：同步 DraftPlan → 写作区章节

### 高级设置
- [ ] 编辑向信息（目标读者/卖点/差异化/前30章承诺）失焦自动保存
- [ ] 世界规则 CRUD + 冲突检测
- [ ] 写法选择绑定/解绑

---

## 写作区

- [ ] 章节列表按卷分组
- [ ] + 卷（直接创建写作区卷章）
- [ ] + 章（追加到最后卷末尾）
- [ ] 删卷（级联删 Chapter + Plan）
- [ ] 删章（删 Chapter + Plan，卷内序号重排）
- [ ] 选章后在 Tiptap 编辑正文
- [ ] AI 单章生成（SSE 流式输出）
- [ ] 自动导演批量生成

---

## 设置页

- [ ] 模型配置：三厂商 API Key 可配可保存
- [ ] 默认模型下拉框 + 测试连接
- [ ] 创作偏好：POV/节奏/基调/题材/章字数/目标章数
- [ ] 偏好保存后新建小说自动填入
- [ ] 页面可滚动

---

## 联动规则（重点）

- [ ] 蓝图增删改 **不影响** 写作区
- [ ] 写作区增删改 **立即同步** 到蓝图草稿
- [ ] 确认按钮：DraftPlan → Chapter，标记 synced，**不删除草稿**
- [ ] 撤销按钮：写作区 → 覆盖蓝图草稿
- [ ] AI 二次生成：只覆盖 synced=true 的草稿，手动编辑保留
